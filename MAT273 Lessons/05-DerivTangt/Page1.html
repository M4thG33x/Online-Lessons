Partial Derivatives

Back in Calc 1, we defined the derivative as the instantaneous rate of change or the slope of the line tangent to a function \(f\) using the following formula, which involved the limit of the difference quotient.

\[f'(x) = \frac{dy}{dx} = \lim_{h \to 0} \frac{f(x+h)-f(x)}{h}\]

Now that we are studying functions of multiple variables, we want to again be able to compute and apply the instantaneous rate of change or tangent line to our multi-variable functions. For now, we will focus on computing the derivative in a direction parallel to the \(x\)-axis denoted by \(f_x\) and a direction parallel to the \(y\)-axis denoted by \(f_y\) for some function \(f(x,y)\).

The following GeoGebra applet illustrates the partial derivatives for a given function. The two checkboxes control whether to display the lines tangent to the surface at the chosen point that are in the \(x\) or \(y\) directions, meaning they would appear parallel to the \(x\) and \(y\) axes, respectively, if you were looking straight down on the \(xy\)-plane from above on the positive \(z\)-axis. You may Move the blue point on the \(xy\)-plane to move the corresponding orange point on the graph at \(z = f(a,b)\). You may click-and-drag to rotate the axes about the origin.

We define the partial derivatives of \(f\) using the formulas below.

\[f_{x}(x,y) = \frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h,y)-f(x,y)}{h}\] \[f_{y}(x,y) = \frac{\partial f}{\partial y} = \lim_{h \to 0} \frac{f(x,y+h)-f(x,y)}{h}\]

Note that the symbol \(\partial\) for the partial derivative are more curled, or more of a script letter, than the \(d\) for the original derivative notation.

Notice the similarities between the limit formula to define the partial derivatives of a function \(f(x,y)\) compared to the limit definition for \(f(x)\). An important thing to observe from these two new formulas is that only one variable is changing in each. So the partial derivative \(f_x\) only looks at how changes in \(x\) cause \(f\) to change and \(f_y\) only looks at how changes in \(y\) cause \(f\) to change. What this means is that \(f_x\) treats \(y\) as a constant and \(f_y\) treats \(x\) as a constant.

Example:

To illustrate this idea, consider a function \(f(x,y)\) given as a table of values. The numbers in the first row are \(x\) values and the numbers in the first column are \(y\) values. The values inside the table are the function values for the corresponding \(x\) and \(y\) values.

Table of \(f(x,y)\) Values
\(y\) vs \(x\) \(x=2\) \(x=5\) \(x=7\)
\(y=1\) 5 8 6
\(y=5\) 6 10 11
\(y=8\) 7 12 15

Let's use the table to approximate \(f_x\) which is the change in the \(x\) direction and \(f_y\) which is the change in the \(y\) direction at the point \((5,5)\).

From the table we see that \(f(5,5) = 10\).

To approximate the value of \(f_{x}(5, 5)\), we could find how \(f\) changes from \(x=2\) to \(x=5\) (which makes \(h=-3\)) and average it with the change from \(x=5\) to \(x=7\) (which makes \(h=2\)).

\[\frac{f(2,5)-f(5,5)}{-3} = \frac{6-10}{-3} = \frac{4}{3}\] \[\frac{f(7,5)-f(5,5)}{2} = \frac{11-10}{2} = \frac{1}{2}\] \[f_{x}(5,5) \approx \frac{\frac{4}{3} + \frac{1}{2}}{2} = \frac{11}{12} \approx 0.9167\]

We see that \(f\) is changing by approximately \(\frac{4}{3}\) from the left (in the \(x\) direction) and that \(f\) is changing by approximately \(\frac{1}{2}\) from the right (in the \(x\) direction). So we can conclude that the values of the function were increasing by approximately \(0.9167\) in the \(x\) direction at the point \((5,5)\).

To approximate the value of \(f_{y}(5, 5)\), we could find how \(f\) changes from \(y=1\) to \(y=5\) (which makes \(h=-4\)) and average it with the change from \(y=5\) to \(y=8\) (which makes \(h=3\)).

\[\frac{f(5,1)-f(5,5)}{-4} = \frac{8-10}{-4} = \frac{1}{2}\] \[\frac{f(5,8)-f(5,5)}{2} = \frac{12-10}{3} = \frac{2}{3}\] \[f_{y}(5,5) \approx \frac{\frac{1}{2} + \frac{2}{3}}{2} = \frac{7}{12} \approx 0.5833\]

We see that \(f\) is changing by approximately \(\frac{1}{2}\) from the left (in the \(y\) direction) and that \(f\) is changing by approximately \(\frac{2}{3}\) from the right (in the \(y\) direction). So we can conclude that the values of the function were increasing by approximately \(0.5833\) in the \(y\) direction at the point \((5,5)\).

Another important result of treating one variable constant while determining the partial derivative of another variable is that all the derivative rules you learned back in Calc 1 still apply. Consider the function \(f(x,y) = 5 x^3 y^3\).

  • The partial derivative of \(f\) with respect to \(x\) is \(f_{x}(x,y) = 15 x^2 y^3\). Notice that we only differentiated the \(x^3\) part, since the 5 and \(y^3\) are just treated as constant multiples.
  • The partial derivative of \(f\) with respect to \(y\) is \(f_{y}(x,y) = 15 x^3 y^2\). Notice that this time we only differentiated the \(y^3\) part, so the 5 and \(x^3\) are just constant multiples.

So no matter how complicated the function make look, we can just use the derivative rules we already know and treat the other variable as a constant.

While we have only consider functions of two variables, the above ideas can be extended to functions of any number of variables. So for \(f(x,y,z)\), we can compute the partial derivatives of \(f\) with respect to \(x\), \(y\), and \(z\), which would be \(f_x\), \(f_y\), and \(f_z\), respectively.

Higher Order Partial Derivatives

Recall from Calc 1 that if the first derivative of a function represents its slope at a point, then its second derivative represents its concavity at a point. Since the 2nd derivative is just the derivative of the 1st derivative function, we can also find the 2nd partial derivatives using the same rules as above. The only extra piece to consider is which variables we are differentiating over - they can get mixed. This means, for example, that we can take the 1st derivative of \(f\) with respect to \(x\) to get \(f_x\), and then we can take the derivatiave of \(f_x\) with respect to either \(x\) or \(y\) to get \(f_{xx}\) or \(f_{xy}\), respectively.
