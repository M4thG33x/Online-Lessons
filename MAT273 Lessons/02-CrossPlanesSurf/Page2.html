Lines & Planes

Just as we built up our toolset of functions in precalculus, starting with lines and moving on to polynomials and other functions in the xy-plane, we need to build up to the point where we can express curves and surfaces in 3D space. We will start in a similar fashion, looking at lines and planes in space.

Lines

Recall from algebra that a line can be expressed by the equation \(y = mx + b\), where \(m\) is the slope and \(b\) is the \(y\)-intercept. Describing a line in 3D space is a little more complicated as it will require using an additional parameter, but the result will be a similar vector equation. We just need a point (position vector) and a slope (direction vector).

Suppose we are g

Given a point \(P(x_0, y_0, z_0)\) with position vector \(\vec{r}_0 = \langle x_0, y_0, z_0 \rangle\) and a direction vector \(\vec{v} = \langle a, b, c \rangle\), the vector equation of the line that passes through the point \(P\) parallel to \(\vec{v}\) is defined by the following equation, for some parameter \(t\) where \(-\infty \lt t \lt \infty\).

\[\begin{align*} \vec{r}(t) &= \vec{r}_0 + \vec{v}t \\ &= \langle x_0, y_0, z_0 \rangle + \langle a, b, c \rangle t \\ &= \langle x_0 + at, y_0 + bt, z_0 + ct \rangle \end{align*}\]

We can also separate the components of the linear vector to get the parametric equations \(x = x_0 + at\), \(y = y_0 + bt\), and \(z = z_0 + ct\).

Planes

In essence, we could think of a line as the set of all points (including the initial point) that are parallel to a given vector. Similarly, we could think of a plane as the set of all points (including the initial point) that are perpendicular to a given vector.

So what does the equation of a plane look like?

Given a point \(P(x_0, y_0, z_0)\) and a normal vector \(\vec{n} = \langle a, b, c \rangle\), the scalar equation of the plane that passes through \(P\) and is perpendicular to \(\vec{n}\) is given by the following formula.

\[a(x - x_0) + b(y - y_0) + c(z - z_0) = 0\]

This can be simplified to the equation \(ax + by + cz = d\) where \(d = ax_0 + by_0 + cz_0\).
