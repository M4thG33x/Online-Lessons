Vector Curves

We have already seen how to graph lines and planes using vectors, so now we want to expand our toolbox to be able to graph more complex functions. We can do this by defining these functions as vectors.

Vector Functions

A vector-valued function has the form \(\vec{r} \left( t \right) = \langle x(t),y(t),z(t) \rangle\) where the components of \(\vec{r}\) are scalar functions of some Real number parameter \(t\).

So each point of a vector function is defined by a vector whose components are parametric equations. See the following GeoGebra examples.

To graph a vector function in GeoGebra, you will use the curve(r(t), t, a, b) command. The a and b values represents the domain interval [a, b] for the variable t. For example, to graph the helix above you would need to enter the expression curve((4cos(t),sin(t),t/2),t,-4π, 4π).

  1. \(\vec{r} = \langle \cos(t), \sin(t), \frac{1}{2} \sin(t) \rangle\)
  2. \(\vec{r} = \langle \lvert t \rvert, \frac{t^2}{5}, t^2-4t \rangle\)

Note that you will need to choose reasonable values for a and b to get a good view of the graph. Play around with different values and see how it affects the graph. Also, to enter absolute value you can either use the command abs(#) or type the vertical bar that is above the backslash key (or "SHIFT + \").

Limits

Now that we can think of functions as vectors, we need to take the next step towards doing calculus with these functions. This involves evaluating limits of vector functions. We can define limits of vector function using language similar to how we defined limits in Calculus I.

The limit of a vector-valued function \(\vec{r}\) is the vector \(\vec{L}\) if \(\vec{r}\) approaches \(\vec{L}\) as \(t\) approaches \(a\). In other words, \[\mathop {\lim} \limits_{t \to a} \vec{r}(t) = \vec{L}.\]

In practical terms, we can evaluate a limit by evaluating the limits of the component functions of the vector function.

\[\lim_{t \to a} \vec{r}(t) = \lim_{t \to a} \langle f(t), g(t), h(t) \rangle = \langle \lim_{t \to a} f(t), \lim_{t \to a} g(t), \lim_{t \to a} h(t) \rangle = \vec{L}\]
