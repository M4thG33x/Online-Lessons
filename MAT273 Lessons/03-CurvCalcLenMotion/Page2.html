Calculus

As we saw on the previous page, we can evaluate the limit of a vector-valued function by evaluating the limits of the separate component functions. If you recall from back in Calculus I, the definition of the derivative and integral both use a limit. So in order to determine the derivative or integal of a vector-valued function, we can do so by determining the derivative or integral of the separate component functions.

Derivatives

Recall from Calculus I that we define the derivative to represent the instantaneous rate of change or slope of the tangent line at each point of the function. This is still true even when using vector-valued functions. The following GeoGebra applet illustrates the tangent vectors of the given curve at any point for a given value of \(t\). You can click-and-drag to rotate the view about the origin. The orange point represents the point on the curve at \(t=2\).

  1. Click on the T checkbox. You should see a purple vector appear at the orange point. This is the tangent vector. Rotate the view around to see how it is tangent to the curve at this point in 3D space.
  2. Check the mx, my and mz checkboxes. As you check each box, you should see another vector added to the orange point. You will have to rotate the view to see all three. These vectors represent the slope vectors of the curve in the directions parallel to the coordinate axes. In other words, they are the component vector of the tangent vector T. So T = mx + my + mz.
  3. Use the slider to move the orange point around. Notice that as it moves and the curvature of the function changes, so does the tangent vector and its component vectors.

For a vector-valued function \(\vec{r}(t) = \langle f(t), g(t), h(t) \rangle\) where \(f\), \(g\) and \(h\) are differentiable functions, the derivative of \(\vec{r}\) is \[\vec{r}'(t) = \langle f'(t), g'(t), h'(t) \rangle.\]

Again, the derivative of a vector-valued function is just the derivatives of its component functions. This same idea holds when determining higher-order derivatives such as \(\vec{r}^{\prime\prime}(t)\) and \(\vec{r}^{\prime\prime\prime}(t)\). Because \(f\), \(g\) and \(h\) are just Real scalar-valued functions of \(t\), we can use all of the derivative rules you learned back in Calculus I.

Because \(\vec{r}\) is a vector and its component functions are scalars, we get the following derivative properties. Let \(\vec{u}\) and \(\vec{v}\) be differentiable vector functions, let \(f\) be a differentiable scalar function, let \(\vec{c}\) be a constant vector, and let \(c\) be a constant scalar.

Vector Function Derivative Rules
Rule Derivative
Constant Vector \(\frac{\partial}{\partial t} \left[ \vec{c} \right] = 0 \)
Constant Multiple \(\frac{\partial}{\partial t} \left[ c\, \vec{u}(t) \right] = c\, \vec{u}'(t) \)
Sum / Difference \(\frac{\partial}{\partial t} \left[ \vec{u}(t) \pm \vec{v}(t) \right] = \vec{u}'(t) \pm \vec{v}'(t) \)
Scalar Product \(\frac{\partial}{\partial t} \left[ f(t) \, \vec{u}(t) \right] = f'(t) \, \vec{u}(t) + f(t) \, \vec{u}'(t) \)
Dot Product \(\frac{\partial}{\partial t} \left[ \vec{u}(t) \cdot \vec{v}(t) \right] = \vec{u}'(t) \cdot \vec{v}(t) + \vec{u}(t) \cdot \vec{v}'(t) \)
Cross Product \(\frac{\partial}{\partial t} \left[ \vec{u}(t) \times \vec{v}(t) \right] = \vec{u}'(t) \times \vec{v}(t) + \vec{u}(t) \times \vec{v}'(t) \)
Chain Rule \(\frac{\partial}{\partial t} \left[ \vec{u} \left( f(t) \right) \right] = f'(t) \, \vec{u}' \left( f(t) \right) \)

Integrals

For a continuous vector-valued function \(\vec{r}(t) = \langle f(t), g(t), h(t) \rangle\), the antiderivative or indefinite integral of \(\vec{r}\) is is the antiderivative of its component functions. In other words, \[\mathop \int \vec{r} dt = \langle \int f(t) dt, \int g(t) dt, \int h(t) dt \rangle .\]

Again, since the component functions \(f\), \(g\) and \(h\) are Real scalar valued functions, we can use the integration rules you learned back in Calculus I & II.
