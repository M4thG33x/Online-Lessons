Motion

Back in Calc I & II, we looked at a scalar position function and thought of the first derivative as velocity and the second derivative as acceleration. So similarly, a vector position function \(\vec{r}(t)\) will have a velocity function \(\vec{v}(t) = \vec{r}'(t)\) with speed \(\lvert \vec{r}'(t) \rvert\) and an acceleration funcction \(\vec{a}(t) = \vec{r}^{\prime\prime}(t)\).

Determining Velocity, Speed, and Acceleration Using a Vector Valued Function
[https://youtu.be/DeWRYCm9cKo]

The reverse is also true. Given an acceleration function \(\vec{a}(t)\), we can find the velocity function \(\vec{v}(t)\) and position function \(\vec{r}(t)\) by integration. If we are given initial values, such as \(\vec{v}(0)\) and \(\vec{r}(0)\), then we can determine the vector constants of integration. Problems of projectile motion may not explicitly state an acceleration function, but we can easily determine the function using a gravity constant, i.e. \(\vec{a}(t) = \langle 0,0,-9.8 \rangle\) m/s2.

Find the Velocity and Position Vector Functions Given the Acceleration Vector Function

Acceleration Components

It can be helpful to break the acceleration vector into its components in the directions of the tangent vector and normal vector, i.e., in the direction the object is traveling and the direction the object is turning.

We can express the vector acceleration function using the acceleration components using the following formula, where \(\vec{T}\) and \(\vec{N}\) are the unit tangent and unit normal vectors, respectively, \(a_{T}\) is the component of acceleration in the direction of motion and \(a_{N}\) is the component of acceleration in the direction of the curvature.

\[\vec{a} = a_{T}\vec{T} + a_{N}\vec{N}\]

There are several formulas for each component, some of which are included in the following table. We typically use the last of each that involves the dot or cross product of velocity and acceleration since they are generally the easiest from a computational perspective.

Acceleration Component Formulas
Tangential Normal
\[a_{T} = {\lvert \vec{v} \rvert}' = \vec{a} \cdot \vec{T} = \frac{\vec{v} \cdot \vec{a}}{\lvert \vec{v} \rvert}\] \[a_{N} = \kappa {\lvert \vec{v} \rvert}^2 = \vec{a} \cdot \vec{N} = \frac{\lvert \vec{v} \times \vec{a} \rvert}{\lvert \vec{v} \rvert}\]
