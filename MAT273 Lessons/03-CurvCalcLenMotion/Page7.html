Computing TNB

When discussing curvature, we defined the unit tangent, unit normal and unit binormal vectors using the formulas \(\vec{T}(t) = \frac{\vec{r}'(t)}{\lvert \vec{r}'(t) \rvert}\), \(\vec{N}(t) = \frac{\vec{T}'(t)}{\lvert \vec{T}'(t) \rvert}\), and \(\vec{B}(t) = \vec{T} \times \vec{N}\), respectively. While we can always compute the TNB functions and TNB values using these formulas, doing so quickly tends to involve tedious derivatives. For example, given the function \(\vec{r}(t) = \langle t, t^2, t^3 \rangle\), we get \(\vec{T}(t) = \frac{\langle 1, 2t, 3t^2 \rangle}{\sqrt{9t^4 + 4t^2 + 1}}\). In order to find \(\vec{N}\), we must first find \(\vec{T}'(t)\) which involves the quotient rule. Doing so for this example wouldn't be too difficult. But if \(\vec{r}\) was a little more complicated, then finding \(\vec{T}'(t)\) would be a lot more complicated. Then once we finally get \(\vec{N}\), we would have to compute a massive cross product to finally get \(\vec{B}\).

So let's outline a process that may be a little easier. Given a smooth curved defined by a parameterized vector function \(\vec{r}\), use the following steps to compute the TNB vectors.

  1. Find the velocity and acceleration functions using \(\vec{v} = \vec{r}'\) and \(\vec{a} = \vec{r}''\).
  2. Compute the unit tangent vector using \(\vec{T} = \frac{\vec{v}}{\left|\vec{v}\right|}\).
  3. Compute the unit binormal vector using \(\vec{B} = \frac{\vec{v} \times \vec{a}}{\left|\vec{v} \times \vec{a}\right|}\).
  4. Compute the unit normal vector using \(\vec{N} = \vec{B} \times \vec{T}\).

Example: Use the process outlined above to compute the TNB values for \(\vec{r}(t) = \langle t, t^2, t^3 \rangle\) at the point \(\left(2, 4, 8\right)\).

  1. Find the velocity and acceleration vectors at the given point.

    \[\vec{v} = \vec{r}'(t) = \langle 1, 2t, 3t^2 \rangle\] \[\vec{a} = \vec{r}''(t) = \langle 0, 2, 6t \rangle\]

    If we compare the given function and point, \(\left(2, 4, 8\right) = \langle t, t^2, t^3 \rangle\), we should observe that this point is reached at \(t = 2\). Plugging 2 into the velocity and acceleration functions results in \(\vec{v} = \langle 1, 4, 12 \rangle\) and \(\vec{a} = \langle 0, 2, 12 \rangle\).

  2. To compute the unit tangent vector at the given point, let's first compute the magnitude of the velocity vector at \(t = 2\).

    \[\left|\vec{v}\right| = \sqrt{1^2 + 4^2 + 12^2} = \sqrt{161}\]

    Now we can compute the unit tangent vector.

    \[\vec{T} = \frac{\vec{u}}{\left|\vec{v}\right|} = \frac{\langle 1, 4, 12 \rangle}{\sqrt{161}}\]

  3. To compute the unit binormal vector, we first need to compute the cross product of velocity with acceleration and then the magnitude of that cross product.

    \[\vec{v} \times \vec{a} = \left| \begin{matrix} \hat{\imath} & \hat{\jmath} & \hat{k} \\ 1 & 4 & 12 \\ 0 & 2 & 12 \end{matrix} \right| = \langle 24, -12, 2 \rangle\] \[\left|\vec{v} \times \vec{a} \right| = \sqrt{24^2 + (-12)^2 + 2^2} = \sqrt{724} = 2 \sqrt{181}\]

    Now we can compute the unit binormal vector.

    \[\vec{B} = \frac{\langle 24, -12, 2 \rangle}{\sqrt{724}} = \frac{\langle 12, -6, 1 \rangle}{\sqrt{181}}\]

  4. To compute the unit normal vector, we just need to compute the cross product of the unit binormal and tangent vectors. We can make this easier by using a property of cross products that allows us to pull any scalar multiples of the vectors out front as a scalar multiple of the cross product.

    \[\vec{N} = \vec{B} \times \vec{T} = \frac{1}{\sqrt{181}} \frac{1}{\sqrt{161}} \left| \begin{matrix} \hat{\imath} & \hat{\jmath} & \hat{k} \\ 12 & -6 & 1 \\ 1 & 4 & 12 \end{matrix} \right| = \frac{\langle -76, -143, 54 \rangle}{\sqrt{181}\sqrt{161}}\]

Notice that in working this example, the only derivatives we ever determined were \(\vec{r}'(t)\) and \(\vec{r}''(t)\). You could check the above results using the original definition formulas. You could also check that our TNB vectors are all orthogonal to each other by making sure each of their dot products are 0.
