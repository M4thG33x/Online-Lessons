Vectors

You have probably already seen vectors in a precalculus or phsycis course, but only in the xy-plane. We want to look at vectors in 3D space. Let's first review vector basics in the plane and then at the end we will extend what we know to space. The following video defines what a vector is and the terminology we use as we work with vectors.

Introduction to Vectors (2D)
[https://www.youtube.com/watch?v=IKzR0Odurm0]

After watching the above video, you should know...

  • The difference between a vector and a scalar. Note that in a typed text we often represent scalar variables as an italicized letter, such as \(x\), and vector variables as a bold letter, such as v. When handwriting a vector variable, we put an arror over the letter, such as \(\vec{v}\). (The arrow may not display properly depending on the browser you are using.)
  • How to compute the magnitude and direction of a vector. Note that magnitude may be represented as either \(|\vec{v}|\) or \(||\vec{v}||\).
  • How to determine the components and express the vector in component form, \(\langle a,b \rangle\).
  • How to use the unit vectors \(\hat{\imath}\) and \(\hat{\jmath}\) to express a vector in \(\hat{\imath}\hat{\jmath}\) form, \(a\hat{\imath} + b\hat{\jmath}\). Note that when handwriting unit vectors, it is common to use a "hat" symbol rather than an arrow. So if \(\vec{v}\) is a unit vector, we could write it as \(\hat{v}\).

2D Vector Operations

There are three vector operations that you should know for now: vector addition, vector subtraction, and scalar multiplication. You are probably already familiar with them, but the following video defines these operations from and algebraic and graphical perspective and gives a few examples. Note that he also discusses a vector dot product, but we will come back to that in the next section.

Vector Operations (2D)
[https://www.youtube.com/watch?v=EYIxFJXoUvA]

One helpful result of scalar multiplication is that is allows us to take any given vector and determine a unit vector that is parallel to it. We can use the following formula.

\[\hat{v} = \frac{\vec{v}}{|\vec{v}|}\]

Vectors in Space

Before we extend the 2D vector concepts we have just covered into 3D space, let's first list the properties of vector operations. Suppose \(\vec{u}\), \(\vec{v}\), and \(\vec{w}\) are vectors and \(a\) and \(c\) are scalars. Note the difference between the scalar \(0\) and the vector \(\vec{0}\), which has no direction and a 0 magnitude.

Properties of Vector Operations
Property Operation
Commutative (addition) \[\vec{u} + \vec{v} = \vec{v} + \vec{u}\]
Associative (addition) \[(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})\]
Additive Identity \[\vec{v} + \vec{0} = \vec{v}\]
Additive Inverse \[\vec{v} + -\vec{v} = \vec{0}\]
Distributive (scalar) \[c(\vec{u} + \vec{v}) = c\vec{u} + c\vec{v}\]
Distributive (vector) \[(a + b)\vec{v} = a\vec{v} + b\vec{v}\]
Zero Scalar (multiplication) \[0\vec{v} = \vec{0}\]
Zero Vector (multiplication) \[c\vec{0} = \vec{0}\]
Multiplicative Identity \[1\vec{v} = \vec{v}\]
Associative (scalar multiplication) \[a(c\vec{v}) = (ac)\vec{v}\]

Also, we already talked about the unit vectors \(\hat{\imath}\) and \(\hat{\jmath}\) in the 2D plane. When we extend this into 3D space, we get the three unit vectos \(\hat{\imath} = \langle 1,0,0 \rangle \), \(\hat{\jmath} = \langle 0,1,0 \rangle \), and \(\hat{k} = \langle 0,0,1 \rangle\).
