Vectors

You have probably already seen vectors in a precalculus or phsycis course, but only in the xy-plane. We want to look at vectors in 3D space. Let's first review vector basics in the plane and then at the end we will extend what we know to space. The following video defines what a vector is and the terminology we use as we work with vectors.

After watching the above video, you should know...

  • The difference between a vector and a scalar. Note that in a typed text we often represent scalar variables as an italicized letter, such as \(x\), and vector variables as a bold letter, such as v. When handwriting a vector variable, we put an arrow over the letter, such as \(\vec{v}\). (The arrow may not display properly depending on the browser you are using.)
  • How to compute the magnitude and direction of a vector. Note that magnitude may be represented as either \(|\vec{v}|\) or \(||\vec{v}||\).
  • How to determine the components and express the vector in component form, \(\langle a,b \rangle\).
  • How to use the unit vectors \(\mathbf{i}\) and \(\mathbf{j}\) to express a vector in \(\mathbf{ij}\) form \(a\mathbf{i} + b\mathbf{j}\). Note that when handwriting unit vectors, it is common to use a "hat" symbol rather than an arrow. So if \(\vec{v}\) is a unit vector, we could write it as \(\hat{v}\).

Evaluating expressions involving scalars and vectors follow properties similar to properities of Real numbers, like the commutative and assotiative properties of addition or the distributive property of addition and multiplication. Here are some of the properties we have for scalar and vector operations.

Properties of Vector Operations
Property
Operation
Commutative (addition)
\[\vec{u} + \vec{v} = \vec{v} + \vec{u}\]
Associative (addition)
\[(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})\]
Additive Identity
\[\vec{v} + \vec{0} = \vec{v}\]
Additive Inverse
\[\vec{v} + -\vec{v} = \vec{0}\]
Distributive (scalar)
\[c(\vec{u} + \vec{v}) = c\vec{u} + c\vec{v}\]
Distributive (vector)
\[(a + b)\vec{v} = a\vec{v} + b\vec{v}\]
Zero Scalar (multiplication)
\[0\vec{v} = \vec{0}\]
Zero Vector (multiplication)
\[c\vec{0} = \vec{0}\]
Multiplicative Identity
\[1\vec{v} = \vec{v}\]
Associative (scalar multiplication)
\[a(c\vec{v}) = (ac)\vec{v}\]

Note that we used several of these properties when solving the application problem in the video above. When adding the three force vectors together, we must technically pick an order to add them. This applies the associative property. When isolating the fourth force vector by subtracting the sum of the given three force vectors, we applied the additive inverse property.
