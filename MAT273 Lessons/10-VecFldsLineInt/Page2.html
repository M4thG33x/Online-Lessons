Line Integrals

Before defining line integrals, we first need to look at how we can parameterize a curve. We originally did this back when talking about arc length in Unit 1. The following video gives a few examples to help refresh your memory.

Defining a Smooth Parameterization of a Path

[https://youtu.be/Vyltj0MpuTU]

Now that we have discussed how to parameterize a curve, we can look at line integrals over scalar functions and over vector fields.

Scalar Functions

We say a function is smooth if it has no sharp corners or cusps. A parameterized curve \(\vec{r}(t)\) is smooth if \(\vec{r}'(t)\) is continuous and \(\vec{r}'(t) \ne 0\). If an entire curve is not smooth but it can be broken down into a finite number of smaller smooth curves, then we say it is piecewise smooth.

For a function \(f\) that is continuous on a (piecewise) smooth curve \(C\) defined by \(\vec{r}(t) = \langle x(t),y(t) \rangle\) for \(a \le t \le b\), the line or path integral of \(f\) over \(C\) is defined below.

\[\underset{C}{\mathop \int}\,f(x,y)~ds = \int_{a}^{b}\,f(x(t),y(t)) \,\vert \vec{r}'(t)\vert~dt\]

The left side of this formula defines a line integral as the integration of the function \(f\) along the curve \(C\) with respect to its arc length, \(ds\). The right side says that we can compute the line integral using any parameterization, \(dt\), which is how we will typically evaluate a line integral. We get the right side from the left side using the arc length formula that we learned about in unit 1, which gives us the differential relationship \(ds = |\vec{r}'(t)|~dt\). Vector Fields

For a vector field \(\vec{F}\) that is continuous over a (piecewise) smooth curve \(C\) defined by \(\vec{r}(t) = \langle x(t),y(t) \rangle\) for \(a \le t \le b\) with unit tangent vector \(\vec{T}\), the line or path integral of \(\vec{F}\) over \(C\) is defined below.

\[\underset{C}{\mathop \int}\, \vec{F} \cdot \vec{T}~ds = \underset{C}{\mathop \int}\, \vec{F} \cdot ~d\vec{r} = \int_{a}^{b}\,\vec{F}(\vec{r}(t)) \cdot \vec{r}'(t)~dt\]

The left side of the formula defines a line integral of a vector field as the integration of the tangential component of the field along the curve with respect to its arc length. We typically express the line integral using the middle part of the formula, which is the result of \(\vec{T}~ds = \vec{T}\,|\vec{r}'(t)|~dt = \frac{\vec{r}'(t)}{|\vec{r}'(t)|}\,|\vec{r}'(t)|~dt = \vec{r}'(t)~dt = d\vec{r}\). The right side of the formula expresses how we actually compute the line integral using any parameterization of the curve. Again, this could easily be extend to a vector field in 3D space.
